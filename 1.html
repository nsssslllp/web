    <생활코딩-html 강의>
  1. 주요태그

    2. <p>

    <br>

    <img>- width,height: 하나만 쓰면 자동 비율조정,alt:깨질때, title:툴팁, src:파일명

    <table>-표묶음, border:외곽선, <tr>:행구분,<td>:항목구분

  2. 속성

    3. 속성: 태그 뒤에 다는 것(속성간 순서 무시)

  3. 하이퍼텍스트

    4. <a>: 링크(하이퍼텍스트 구현) 속성: href, title, target..

    예시) <a href="링크복붙" title="전설적인 프로그래머" target="_blank">도널드머시기)

    target="_blank" : 새창에 띄움

    title : 마우스 대면 작은 툴팁 뜸

  4. 목록 (태그의중첩)

    5. <list>를 ul태그 혹은 ol태그로 감싼다
    <head> : 문서전체 꾸미는 부가적 정보를 감싼다
    <body>: 문서의 본문내용을 감싼다
    <html> : 맨위에 쓴다

  5. 입력양식-form

    6. - <form> : 사용자가 입력한 정보 서버로 전송할 때 사용 (form으로 묶음)
    action : 보낼 서버의 주소

    -텍스트 입력 <input> : 사용자가 입력가능한 칸이 생성
    type : text, password, submit
    name: url에 사용자가 입력한 정보 포함됨
    value : default값
    submit: 제출버튼 (value=" "쓰면 submit대신 그 문자로 나옴)
    radio,checkbox

    예시)
    *id에 pjs1638입력하면 url에 id=pjs1638이 포함됨
    <form action="http://localhost/login.php">
    <p>아이디 : <input type="text" name="id"></p>
    <p>비밀번호 : <input type="password" name="pwd"></p>
    <p>주소 : <input type="text" name="address"></p>
    <input> type="submit">

    <textarea>: 여러줄 입력받을때
    cols,rows : 텍스트입력란 크기 조정 가능(선택사항)
    태그 사이에 입력한 문자가 default value가 됨 (→value에 여러줄 입력 시 사용)

    </form>
